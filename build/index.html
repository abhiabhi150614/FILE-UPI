For Regular Users

ğŸ“± Simple Setup (2 minutes)

  • Smart Reading: System reads text from PDFs and images automatically. Search "hospital bill October" and find it even if it's a photo.

  • Family Sharing: Share entire folders with family (like shared bank accounts). Useful for medical records, household bills.
  • Works Offline: Mobile app works without internet. Syncs when connection returns.

Real-World Usage Scenarios

Scenario 1: Healthcare Journey

Problem: You visit multiple doctors. Each gives reports. You lose some, can't find others during emergencies.

With FileFlow:

  1. Doctor's clinic scans QR on your phone, sends reports to your "Hospital Reports" folder instantly.
  2. Reports contain: doctor name, date, hospital, type - all searchable.
  3. 3 years later, new doctor asks for old blood tests. You search "blood test 2022" - found in 2 seconds.
  4. You share temporary access with new doctor. They review, access expires after 7 days automatically.
  5. During emergency, you grant access to ER doctor - they see complete medical history immediately.

Scenario 2: Tax Time Made Easy

Problem: Tax season = chaos. Finding receipts, bills, invoices from entire year.

With FileFlow:

  1. Throughout year, every bill/receipt gets scanned and sent to your "Bills" folder.
  2. Come tax time: Search "bills 2024" or filter by date range.
  3. Export all as ZIP with organized folders.
  4. Share secure link with CA - they access only what they need.
  5. Next year? All history is there. Comparison becomes easy.

Scenario 3: Professional Document Trail

Problem: Companies send important documents via email. Years later, you need offer letter - can't find it.

With FileFlow:

  1. Company HR sends offer letter to your "Company" folder with transaction ID.
  2. System keeps: timestamp, company email, document version.
  3. You accept offer, HR sends updated contract - linked to original.
  4. 5 years later, new job asks for previous offer - you find it instantly with complete verification trail.
  5. You can prove: "Company X sent me this on [date]" - complete audit trail.

Key Architectural Decisions

ğŸ“¦ Microservices vs Monolith

Decision: Modular Monolith initially, split to microservices at scale
Why: Faster development, easier debugging. Can extract to services later (upload service, OCR service, etc.)

ğŸ—„ï¸ Database Strategy

PostgreSQL Primary: ACID compliance, JSON support (JSONB), full-text search, proven at scale
Why not NoSQL: Need strong consistency for audit trails and transactions

ğŸ“ File Storage

S3 (or compatible): Scalable, durable, built-in encryption
Strategy: Presigned URLs for direct upload/download (reduces backend load)

ğŸ” Search Strategy

ElasticSearch: For OCR text, full-text search, complex queries
PostgreSQL: For simple metadata searches
Hybrid: Best of both worlds

Indexes & Performance

  • B-tree indexes: On foreign keys, status fields, timestamps for fast lookups
  • GIN indexes: On JSONB fields, arrays (tags), full-text search (ocr_text)
  • Partial indexes: Where clauses on deleted_at, status to reduce index size
  • Compound indexes: (user_id, created_at DESC) for timeline queries

from datetime import datetime, timedelta
from jose import JWTError, jwt
from passlib.context import CryptContext

pwd_context = CryptContext(schemes=["bcrypt"], deprecated="auto")

SECRET_KEY = "your-secret-key-from-env"
ALGORITHM = "HS256"
ACCESS_TOKEN_EXPIRE_MINUTES = 30

def verify_password(plain_password: str, hashed_password: str) -> bool:
    return pwd_context.verify(plain_password, hashed_password)

def get_password_hash(password: str) -> str:
    return pwd_context.hash(password)

def create_access_token(data: dict, expires_delta: timedelta = None):
    to_encode = data.copy()
    expire = datetime.utcnow() + (expires_delta or timedelta(minutes=15))
    to_encode.update({"exp": expire, "type": "access"})
    return jwt.encode(to_encode, SECRET_KEY, algorithm=ALGORITHM)

def create_refresh_token(data: dict):
    to_encode = data.copy()
    expire = datetime.utcnow() + timedelta(days=7)
    to_encode.update({"exp": expire, "type": "refresh"})
    return jwt.encode(to_encode, SECRET_KEY, algorithm=ALGORITHM)

# app/services/storage.py

import boto3
from botocore.config import Config

class S3StorageService:
    def __init__(self):
        self.s3_client = boto3.client(
            's3',
            aws_access_key_id=settings.AWS_ACCESS_KEY_ID,
            aws_secret_access_key=settings.AWS_SECRET_ACCESS_KEY,
            region_name=settings.AWS_REGION
        )
        self.bucket = settings.S3_BUCKET
    
    def generate_storage_key(self, user_id: str, filename: str) -> str:
        """Generate unique S3 key"""
        timestamp = datetime.utcnow().isoformat()
        unique_id = hashlib.sha256(
            f"{user_id}{filename}{timestamp}".encode()
        ).hexdigest()[:16]
        return f"users/{user_id}/files/{unique_id}/{filename}"
    
    def create_presigned_upload_url(
        self,
        storage_key: str,
        content_type: str,
        expires_in: int = 3600
    ) -> str:
        """Generate presigned URL for direct upload"""
        url = self.s3_client.generate_presigned_url(
            'put_object',
            Params={
                'Bucket': self.bucket,
                'Key': storage_key,
                'ContentType': content_type,
            },
            ExpiresIn=expires_in
        )
        return url
    
    def create_multipart_upload(
        self,
        storage_key: str,
        content_type: str
    ) -> dict:
        """Initialize multipart upload for large files"""
        response = self.s3_client.create_multipart_upload(
            Bucket=self.bucket,
            Key=storage_key,
            ContentType=content_type,
            ServerSideEncryption='aws:kms'
        )
        return {
            'upload_id': response['UploadId'],
            'storage_key': storage_key
        }
    
    def create_presigned_download_url(
        self,
        storage_key: str,
        expires_in: int = 3600
    ) -> str:
        """Generate presigned URL for download"""
        url = self.s3_client.generate_presigned_url(
            'get_object',
            Params={'Bucket': self.bucket, 'Key': storage_key},
            ExpiresIn=expires_in
        )
        return url

            'owner_id': file_data['owner_user_id'],
            'filename': file_data['filename'],
            'ocr_text': file_data.get('ocr_text', ''),
            'folder_id': file_data['folder_id'],
            'sender_id': file_data.get('sender_id'),
            'tags': file_data.get('tags', []),
            'mime_type': file_data['mime_type'],
            'created_at': file_data['created_at'],
            'metadata': file_data.get('extracted_metadata', {})
        }
        
        await self.es.index(
            index=self.index_name,
            id=file_id,
            document=document
        )
    
    async def search_files(self, user_id: str, query: str, filters: dict = None):
        """Search files with filters"""
        search_query = {
            "query": {
                "bool": {
                    "must": [
                        {
                            "multi_match": {
                                "query": query,
                                "fields": [
                                    "filename^3", 
                                    "ocr_text^2",
                                    "tags^2",
                                    "metadata.*"
                                ]
                            }
                        }
                    ],
                    "filter": [
                        {"term": {"owner_id": user_id}}
                    ]
                }
            }
        }
        
        # Add additional filters
        if filters:
            for key, value in filters.items():
                if value:
                    search_query["query"]["bool"]["filter"].append(
                        {"term": {f"{key}.keyword": value}}
                    )
        
        result = await self.es.search(
            index=self.index_name,
            body=search_query
        )
        
        return [hit['_source'] for hit in result['hits']['hits']]

# app/services/ocr_service.py

import pytesseract
from PIL import Image
import pdf2image

class OCRService:
    def __init__(self):
        self.supported_formats = ['.pdf', '.jpg', '.jpeg', '.png', '.tiff']
    
    async def extract_text(self, file_path: str, mime_type: str) -> str:
        """Extract text from file using OCR"""
        try:
            if mime_type == 'application/pdf':
                return await self._process_pdf(file_path)
            elif mime_type.startswith('image/'):
                return await self._process_image(file_path)
            else:
                return ""
        except Exception as e:
            logger.error(f"OCR processing failed: {e}")
            return ""
    
    async def _process_pdf(self, file_path: str) -> str:
        """Extract text from PDF"""
        images = pdf2image.convert_from_path(file_path)
        full_text = ""
        
        for page_num, image in enumerate(images):
            text = pytesseract.image_to_string(image)
            full_text += f"--- Page {page_num + 1} ---\n{text}\n"
        
        return full_text
    
    async def _process_image(self, file_path: str) -> str:
        """Extract text from image"""
        image = Image.open(file_path)
        return pytesseract.image_to_string(image)
